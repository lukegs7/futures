# vue3.0 Composition API 入门教程

光阴荏苒，vue3.0已经进入了 beta 版本了。很多朋友说，你该出个系列的教程了，毕竟当年我写得 vue 2.0 的系列的博文在网上影响并不小。那就写吧，我力求简洁明了的把新的知识点都覆盖到。

## 安装以及运行

```bash
# clone 项目
git clone https://github.com/fengcms/vue3-demo
# 进入项目文件夹
cd vue3-demo
# 安装依赖
npm i
# 运行开发环境
npm run dev
# 编译代码
npm run build
```
